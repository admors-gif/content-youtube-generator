# Wellness audio beds

Curated instrumental beds for autohipnosis and long meditation formats.

Rules:
- Instrumental only: no vocals, no lyrics, no sudden changes.
- Use only tracks with clear commercial rights.
- Store production assets as normalized MP3 loops, not raw WAV exports.
- Keep filenames ASCII and reference them from `manifest.json`.

The text pipeline selects a track from `manifest.json` by topic/personalization
intent and writes it into `autohipnosis.background_music` or
`longMeditation.background_music`. The factory loops the selected asset under
the voice and fades the full bed in/out during assembly.

If no curated track is available, long meditation can still fall back to the
local procedural ambient bed generated by FFmpeg.

# Esto No Es Amor — Kit De Identidad

## Archivos

- `banner-2560x1440.png`: banner recomendado para YouTube.
- `logo-1024x1024.png`: avatar/logotipo cuadrado.
- `youtube_channel_description.txt`: descripcion SEO lista para pegar en YouTube Studio.
- `brand-kit.json`: paleta, motivos, prohibiciones y reglas de thumbnail.
- `thumbnail_prompt_system.md`: prompt base para mantener identidad visual.

## Uso

Sube el banner y logo manualmente en YouTube Studio. Mantén el texto del canal
en la descripcion principal y usa las reglas del prompt para futuros thumbnails.

La identidad busca que el canal se reconozca por objetos emocionales, luz ambar,
sombras teal y composiciones intimas sin rostros ni manos.
